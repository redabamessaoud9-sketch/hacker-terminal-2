# Matrix Terminal Demo

Un petit projet HTML/CSS/JavaScript inspiré des interfaces de science-fiction.

## Fonctionnalités

- 🟢 Pluie de caractères façon Matrix
- ⌨️ Faux terminal animé
- 📊 Barre de progression
- 🔊 Petits effets sonores générés par le navigateur
- 😂 Message final indiquant clairement qu'il s'agit d'une animation

## Utilisation

1. Télécharger le projet
2. Ouvrir `index.html`

Ou le publier avec GitHub Pages.
